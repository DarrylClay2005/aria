# Aria Stability Layer

Aria owns RECOVER/doctoring and now has swarm-wide recovery pressure/degraded-mode controls. Active cooldowns are stored in `aria_swarm_recovery_cooldowns`.

Key env knobs: `ARIA_SWARM_RECOVERY_WINDOW_SECONDS`, `ARIA_SWARM_RECOVERY_MAX_PER_GUILD`, `ARIA_SWARM_RECOVERY_MAX_GLOBAL`, `ARIA_SWARM_DEGRADED_COOLDOWN_SECONDS`.
